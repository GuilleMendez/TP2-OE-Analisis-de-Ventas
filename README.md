El escenario elegido fue el siguiente:

Análisis de Ventas de una Pequeña Empresa
El equipo trabajará con un conjunto de datos simulados de ventas comerciales.

Objetivo
Analizar la información de ventas para generar indicadores básicos que permitan interpretar
el desempeño de la empresa.
Tareas sugeridas
● Importar un archivo de ventas con información de:
○ producto
○ cantidad vendida
○ precio
○ fecha de venta
● Calcular indicadores como:
○ ventas totales
○ producto más vendido
○ ventas por mes
● Generar un gráfico simple que represente la evolución de ventas
